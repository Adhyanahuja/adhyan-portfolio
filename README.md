# Adhyan Ahuja — Portfolio

A premium dark-themed portfolio website built for MBA applications to top business schools.

**Live**: [adhyan-portfolio.vercel.app](https://adhyan-portfolio.vercel.app)

## Tech Stack

- **Framework**: Next.js 16 (App Router, Turbopack)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion
- **Fonts**: Inter, Space Grotesk, Caveat
- **Deployment**: Vercel

## Features

- Dark cinema aesthetic with glassmorphism UI
- Animated sticky notes "My Arsenal" skills collage
- Cinematic scroll animations and image reveals
- Interactive timeline with company metrics
- Publications section with Springer & IEEE research
- Company logo marquee (D.E. Shaw, Piramal, ITC, CSIR)
- Smooth scrolling with Lenis
- Gradient cursor tracking
- Fully responsive design

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
